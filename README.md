# GW-Exchanger
да
Сервис обмена валют (Currency Exchange Service) для системы электронных кошельков.

## Описание

GW-Exchanger - это микросервис, предоставляющий API для получения курсов валют. Сервис реализует gRPC-интерфейс для получения текущих курсов обмена между различными валютами.

## Функциональность

- Получение курсов обмена всех доступных валют
- Получение курса обмена для конкретной пары валют
- Хранение курсов обмена в PostgreSQL базе данных

## Архитектура

- Язык программирования: Go
- Протокол взаимодействия: gRPC
- База данных: PostgreSQL
- Контейнеризация: Docker

## Конфигурация

Сервис использует следующие параметры конфигурации:
- Порт gRPC сервера (по умолчанию: 50052)
- Параметры подключения к PostgreSQL (хост, порт, имя пользователя, пароль, база данных)

## Структура проекта

- `cmd/main.go` - точка входа в приложение
- `internal/` - внутренняя реализация сервиса
- `proto/` - protobuf-определения интерфейсов
- `pkg/` - общие пакеты и утилиты
- `Dockerfile` - инструкции для сборки контейнера

## Использование

Сервис предоставляет два gRPC метода:
1. `GetExchangeRates` - возвращает все доступные курсы обмена
2. `GetExchangeRateForCurrency` - возвращает курс обмена для указанной пары валют

## Установка и запуск

Сервис может быть запущен как отдельное приложение или в составе Docker-контейнера с использованием docker-compose.
